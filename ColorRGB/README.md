# RGB 색상 구조체

RGB 색상을 나타내는 `ColorRGB` 구조체를 만들고, 색상의 밝기를 계산하는 
프로그램을 작성하세요.

**요구사항:**

1. `ColorRGB` 구조체를 정의하세요.
   - `R`, `G`, `B` 필드 (int 타입, 0~255 범위)
   - 세 값을 받아 초기화하는 생성자
   - `GetBrightness()` 메서드: 밝기를 반환 (R + G + B) / 3 (int)

2. 세 가지 색상을 생성하세요.
   - 빨강: (255, 0, 0)
   - 흰색: (255, 255, 255)
   - 커스텀: (100, 150, 200)

3. 각 색상의 RGB 값과 밝기를 출력하세요.

## 예상 실행 결과

```
=== RGB 색상 밝기 계산 ===
빨강 - R: 255, G: 0, B: 0 → 밝기: 85
흰색 - R: 255, G: 255, B: 255 → 밝기: 255
커스텀 - R: 100, G: 150, B: 200 → 밝기: 150
```
